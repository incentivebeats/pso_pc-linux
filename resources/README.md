These are resources that can help you get started with setting up Phantasy Star Online PC.  

1. D3D8.dll and dgVoodoo config
Disclaimer: This was created with LLM assistance using Claude. This is a D3D8 file intended for widescreen that tries to both address issues with culling and draw distance. The default dgVoodoo settings are for 14:9 as that works best with 2D space.
